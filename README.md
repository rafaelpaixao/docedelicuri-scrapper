# Doce de Licuri - Scrapper

#### Example

1. Get options

```
yarn extract-options
``` 

2. Get payments for Feira de Santana

```
yarn extract-payments 2910800
```

3. Generate payments meta

```
yarn validate-payments
``` 

4. Clean payments

```
yarn clean-payments
``` 

5. Generate monthly reports

```
yarn make-reports
```
